## Prerequisites
Node version 18.17.x
## Setup .env file

```bash
DATABASE_URL=

JWT_SECRET=

NEXT_PUBLIC_APP_URL=http://localhost:3000
NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME=
NEXT_PUBLIC_CLOUDINARY_UPLOAD_PRESET=
NEXT_PUBLIC_ONE_SIGNAL_APP_ID=

MAILERSEND_API_KEY=
FROM_EMAIL=
```

## Setup Prisma
Add PostgreSQL Database

```bash
npx prisma db push
```

## Start to dev

```bash
yarn dev
```
